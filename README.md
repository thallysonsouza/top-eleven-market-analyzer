# Top Eleven Market Analyzer

Ferramenta para análise de transferências de jogadores do Top Eleven.

## Recursos

- Dashboard Power BI
- Simulador de Leilão em Excel
- Cálculo de preço justo
- Sistema de Score
- Classificação automática de oportunidades

## Arquivos

### Power BI
Dashboard interativo para análise dos jogadores.

### Excel
Simulador para avaliar se vale a pena comprar um jogador no mercado.
